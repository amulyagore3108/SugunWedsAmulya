# Amulya & Sugun's Wedding Invitation

A fun, interactive wedding invitation website with scrollable sections and animations.

## Features

- **Scrollable Design**: Smooth scrolling through different sections
- **Fun Theme**: Colorful gradients, animations, and confetti effects
- **Interactive Elements**: Animated cards, confetti on scroll, and RSVP form
- **Responsive**: Works on desktop and mobile devices

## How to View

1. Open `index.html` in your web browser
2. Or serve it with a local server:
   - If you have Python installed: `python -m http.server 8000`
   - If you have Node.js: `npx serve .`
   - Then open `http://localhost:8000` in your browser

## Customization

- Edit `index.html` to change the wedding details
- Modify `css/style.css` to adjust colors and styling
- Update `js/script.js` for additional interactivity
