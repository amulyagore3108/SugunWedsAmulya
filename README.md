📸 **Place your background image here**

To enable the floral hero background, add a file named **`floral-bg.png`** in this folder.

✅ Expected path (relative to `index.html`):
`images/floral-bg.png`

If you need a sample floral background, you can download one and rename it to `floral-bg.png` (keep it in this folder).

Once placed, the hero section will automatically use it as the background.
